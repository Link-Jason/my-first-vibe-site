# 🕹️ SHALL WE PLAY A GAME? (Retro Pong Vibe)

An authentic, CRT-styled Pong clone built for both **Desktop** and **Mobile** play. This project focuses on "vibe-heavy" aesthetics, featuring scanlines, screen shakes, and glitch effects.

## 🚀 Live Demo
Check out the game here: [https://my-first-vibe-site.vercel.app/](https://my-first-vibe-site.vercel.app/)

## 🕹️ Controls
* **Desktop:** Use the `W` / `S` keys or the `Up` / `Down` arrows to move your paddle.
* **Mobile:** **Drag horizontally** (left to right) across the screen. 
    * *Why?* This allows you to control the paddle without your hand blocking the game view!

## ✨ Features
* **CRT Overlay:** Simulated scanlines and flickering for that 1980s terminal feel.
* **Adaptive AI:** The computer opponent gets faster and smarter as you score points.
* **Glitch Effects:** Visual "breaks" and screen shakes occur during high-velocity impacts and scoring.
* **Responsive Design:** A virtual 3:2 aspect ratio engine that works on any screen size.

## 🛠️ Tech Stack
* **HTML5 / CSS3:** Custom animations and flexbox layout.
* **Vanilla JavaScript:** High-performance `requestAnimationFrame` game loop.
* **Deployment:** Hosted via Vercel with GitHub integration.
